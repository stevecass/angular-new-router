This is the "detail" component.

This is the template for the detail component. When angular switches to an ngNewRouter route which refers to a component called "detail", Angular will by default request /components/detail/detail.html from the web server. It will also, try to construct a javascript DetailController. In the context of this app, the detail component (detail is just a name) will show the detail of one kitty. The DetailController (defined in app/assets/javascripts/controllers.js binds one cat to a variable called detail.cat. This is available in this template as "detail.cat")
